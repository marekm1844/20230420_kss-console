## Description

Project:
This project is a simple example of how to use NestJS with Websockets and Binance API for real-time candle data.
Presented on Knowledge Sharing session at 2023-04-20 for Startup House.
This project was written with AI assistance.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
